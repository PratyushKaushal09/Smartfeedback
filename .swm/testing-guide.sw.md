---
title: Testing Guide (Vitest)
---

This project uses Vitest for unit and integration tests.

## Commands

- Run tests: `pnpm test`
- Type-check: `pnpm typecheck`

## Tips

- Place tests near the code or under a `__tests__` folder.
- Keep tests deterministic and fast.
